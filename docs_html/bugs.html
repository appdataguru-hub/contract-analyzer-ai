#1 BM25Cache (КРИТИЧЕСКИЙ)

Было: BM25Retriever пересоздавался при КАЖДОМ HTTP-запросе (~100-500ms).

Стало:

class BM25Cache:
    def get_or_build(self, collection_name, documents, k):
        if collection_name in self._cache:
            return self._cache[collection_name]  # cache hit
        retriever = BM25Retriever.from_documents(documents)
        self._cache[collection_name] = retriever
        return retriever
